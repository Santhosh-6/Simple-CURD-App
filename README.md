## вљ™ Installation

Clone repository:

```bash
git clone https://github.com/Santhosh-6/Simple-CURD-App.git
```

Move into project directory:

```bash
cd Simple-CURD-App
```

Install backend dependencies:

```bash
cd server
npm install
```

Install frontend dependencies:

```bash
cd client
npm install
```

---

## Configure Environment Variables

Create a `.env` file in the server folder:

```env
MONGO_URI=your_mongodb_connection_string

PORT=5000
```

---

## в–¶ Running the Project

Start backend:

```bash
cd server
npm start
```

Start frontend:

```bash
cd client
npm start
```

Application runs on:

Frontend:

```bash
http://localhost:3000
```

Backend:

```bash
http://localhost:5000
```
